# COVID-19 Annual Statistics in Ibero-American Countries (2019-2024)

[![DOI](https://img.shields.io/badge/DOI-10.7910/DVN/6WISDO-blue)](https://doi.org/10.7910/DVN/6WISDO)

## Overview

Interactive dashboard: COVID-19 Annual Statistics for 21 Ibero-American countries (2019-2024). Confirmed cases, deaths, recoveries, vaccinations, and case fatality rate. DOI: 10.7910/DVN/6WISDO
